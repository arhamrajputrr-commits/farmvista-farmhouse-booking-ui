# 🌿 FarmVista

A modern and responsive farmhouse listing website built with HTML5 and CSS3.

## ✨ Features

- Responsive Design
- Modern Hero Section
- Farm House Listings
- Search Section
- Services Section
- Gallery
- Customer Reviews
- Contact Form
- Smooth Hover Animations
- Mobile Friendly Layout

## 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid

## 🚀 Future Improvements

- Laravel Backend
- MySQL Database
- Booking System
- Admin Dashboard
- User Authentication
